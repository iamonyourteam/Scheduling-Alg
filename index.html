Schedule Demo

Overview

Problem

Given a series of phone calls. Determine the most efficient scheduling strategy for allocating X number of skilled technicians over a 5-day period, taking into account factors such as technician availability, task proficiency, and proximity of customer locations.

Solution

This code is a set of instructions for a computer that helps to assign jobs to people with certain skills. The code first retrieves job information from a file called 'jobs.json' and information about people with skills from a file called 'techs.json'. Then it goes through each job, one by one and check if the person has the required skills for the job, also it checks if the person has enough time to work on the job and if the distance between the job location and the first job assigned to the person is less than 15 minutes, if no jobs are found less then 15 min then the job will check if less then 30 min. If all the conditions are met, the job is assigned to the person. If a job cannot be assigned to a person, it is added to a special list called "overflow". At the end, the code shows the final list of jobs assigned to each person and the list of jobs that couldn't be assigned.
